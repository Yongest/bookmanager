apt-get install nginx 时出现Unable to locate package
sudo apt-get update， 这命令作用是：同步 /etc/apt/sources.list 和 /etc/apt/sources.list.d 中列出的源的索引，这样才能获取到最新的软件包。
    ```
        /usr/sbin/nginx  #主程序
        /etc/nginx   #存放配置文件
        /usr/share/nginx  #存放静态文件
        /var/log/nginx  #存放日志
    ```
